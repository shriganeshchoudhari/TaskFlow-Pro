# `.ai/` — AI-assisted development kit

This folder contains **compact, operational** project context intended for:

- Fast onboarding (humans and AI tools)
- Consistent implementation workflows (feature/bugfix/refactor)
- Repeatable quality gates (tests, lint, CI)
- Decision tracking (ADRs)

Source-of-truth product/technical docs live in `docs/`. This folder focuses on **how to work** in the repo.

Start here:

- `./CONTEXT.md` — one-page project context + commands
- `./REPO_MAP.md` — what lives where
- `./WORKFLOWS.md` — step-by-step dev workflows
- `./QUALITY_GATES.md` — what must pass before merging
- `./DEFINITION_OF_DONE.md` — what “done” means
- `./CHECKLISTS.md` — security/release checklists
- `./PROMPT_TEMPLATES.md` — reusable prompts for PRD/TTD/API/UI changes
- `./adr/` — architecture decision records
